Le parcours WEB c'est quoi?

Le parcours Web permet à l'etudiant d'apprendre à developper son propre site web et de le mettre en ligne

Le parcours Web permet à l'etudiant d'apprendre à developper son propre site web et de le mettre en ligne Le programme convient trés bien à ceux qui ont des idéés de site web et qui veulent passer à l'action et conctrétiser leurs idéés Les technologies utilisés dans ce parcours sont les technologies javascript(React,Node...).Ces technologies sont de plus en plus utiliséés dans le web et permettent de developper des site web trés intuitifs et user friendly.
